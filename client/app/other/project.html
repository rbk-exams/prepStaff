Project

About

Together with a partner, you will work on a project together. You will have 2 days to finish the project and it must fulfill the requirements below.

Project Ideas

  1. Calculator A calculator can be as complex as you want!

  2. Store Your score will have various items for sale and the user can choose what they want and 'purchase'.

  3. Recipe Generator The user chooses the ingredients they have and your website gives back a recipe.

  4. Book Suggestions The user chooses what genres they like and your website gives some suggestions.

  5. Quiz Page! Create a quiz for users.

  6. Basic To Do List The user can add, complete, and remove tasks.

  7. Choose your own! But please let one of the instructors know.

Requirements

The project should make use of everything we have so far touched in this class.

This includes the following:

  • images
  • DOM manipulation and using the document object
  • use of objects
  • use of arrays
  • use of CSS
  • use of abstraction concepts
  • staying DRY

What We Look At

We will not compare your project to someone else's project. We will look at how much progress you have made and if you push yourself.

Do you only stay with things that you are comfortable with? Do you try to learn more?

This is a chance for you to think about what you really want to make and to try to make it!
